# Distribution

## Distribution Channels

| Channel | Status | Notes |
| --- | --- | --- |
| GitHub Releases (macOS DMG) | Repo-visible prerelease | Ships `.dmg` artifacts as the current repo-visible prerelease channel. |
| GitHub Releases (Windows EXE) | Implemented | Builds `.exe` artifacts, but no public Windows artifact is proven yet. |
| GitHub Releases (Linux AppImage + .deb + .rpm) | Implemented | Builds `.AppImage`, `.deb`, and `.rpm` artifacts. |
| Mac App Store | Implemented | Uses the App Store `.pkg` channel. |
| Homebrew Cask | Planned | Follows the release artifact feed. |

Artifacts include .dmg, .exe, .AppImage, .deb, .rpm, .pkg.
