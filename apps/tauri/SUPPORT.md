# Support

Lorvex is still pre-release. Non-security support is best-effort, with the
public support page available at https://lorvex.app/support/.

Use these channels:

- Security vulnerabilities: open a
  [private GitHub security advisory](https://github.com/boyugou/ai-native-todo/security/advisories/new).
  This is the only reporting path — Lorvex has no security email mailbox.
  Do not open a public issue for vulnerabilities.
- Bugs, feature requests, setup questions, troubleshooting, and architecture
  questions: open a GitHub Issue at
  <https://github.com/boyugou/lorvex/issues>.
- Documentation questions: start with [docs/INDEX.md](docs/INDEX.md).
