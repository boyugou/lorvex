# Sharing Analytics and Diagnostics with Apple

Source: [Share analytics, diagnostics, and usage information with Apple](https://support.apple.com/108971)

Last verified: 2026-07-10

## Apple Contract

Apple devices expose an Analytics & Improvements setting through which users
can allow diagnostic and usage information to be sent to Apple and, where the
related option is enabled, shared with app developers.

## Lorvex Mapping

This is system behavior outside Lorvex's own data pipeline. It does not turn
Lorvex's local MetricKit store into developer collection for the App Store
privacy label, but it makes “never transmitted anywhere, not to Apple” too
broad unless the sentence is explicitly limited to the copy persisted by
Lorvex itself.
